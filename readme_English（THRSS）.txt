　Question:　Key configuration possible?
　Answer:　Launch the "config.exe" in the "THRSS_trial0.01b" folder.
　　　　　Select the key field of the tab.
　　　　　"左"			Light button
　　　　　"右"			Right button
　　　　　"上"			UP button
　　　　　"下"			DOWN button
　　　　　"決定"		Decision button
　　　　　"キャンセル"	Cancel button
　　　　　"ショット"	Shot button
　　　　　"スペル"		Spell button
　　　　　"低速移動"	Low-speed movement button
　　　　　"ユーザ定義1"	Special 1.　The default setting "C"
　　　　　"ユーザ定義2"	Special 2.　The default setting "V"
　　　　　"ポーズ"		Pause button

　Question:　Processing heavy.
　Answer:　From the game menu, select the Option.
　　　　　Please increase the number of cut effect.
　　　　　Larger the value, I will reduce the effects of many.

　Question:　Replay does not work properly.
　Answer:　When a file with the same name already exists, you will not be able to play normally.

　Question:　I can not clear Lunatic!
　Answer:　From the game menu, select the Option.
　　　　　Please ON the AutoBomb.
　　　　　When you've hit, I will trigger the SpellCard automatically.
